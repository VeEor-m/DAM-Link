小屏（手机）布局方案

375px 视口宽度下的 3 种主流方式。点选你想要的，下方 mockup 是按 1:1 比例画的。

A. 抽屉 + 底部 Sheet

☰ DAM-Link
详情（半高）

机制：☰ 打开左侧筛选抽屉；点卡片 → 底部 Sheet 滑出详情（半高，可拉成全屏）

手感：最接近原生 App；缺点是 2 层深度（先看列表再展开详情）

B. 顶部 Tab 切换

DAM-Link
资产
筛选
详情

机制：顶部 3 个 Tab（资产 / 筛选 / 详情），点哪个显示哪个面板

手感：极简；缺点是失去上下文（看详情时看不到列表）

C. 自适应 3 栏（图标条）

DAM-Link

机制：左侧 40px 图标条（macOS Finder 风格）= Sidebar；右侧主区 + 详情同列上下排

手感：常驻导航；缺点是图标条在 375px 下吃 11% 宽度，且不如 A 直观
