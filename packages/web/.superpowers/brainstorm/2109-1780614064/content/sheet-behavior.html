手机上的详情 Sheet 行为

点卡片后，详情怎么呈现？两种主流模式。

B1. 半高 Peek + 可拖拽展开

DAM-Link
sunset.png

机制：点卡片 → 半高 sheet 滑出（190px / 340px），拖顶部条可拉到 90% 高

优点：iOS/Android 原生范式；可保留背景上下文（看到列表）。缺点：需要 drag 手势

关闭：拖下 / 点外部 / Esc（如果接键盘）

B2. 点击即全屏

← 资产详情
sunset.png

机制：点卡片 → 全屏详情页（带 ← 返回箭头）

优点：实现最简（路由 push/pop 模式）；空间充足。缺点：失去列表上下文

关闭：← 返回 / 系统返回手势

回 B1 / B2 / Other。
