大屏（>1280px）怎么调？

1920px+ 显示器上当前 3 栏看起来挺空。3 种处理方式。

W1. 啥也不动

机制：1920px 显示器上仍是 160/200px 固定侧栏，中心网格用 flex 自动加列

优点：完全不改。缺点：侧栏窄、详情信息少，大屏浪费

W2. 加宽详情面板

详情

机制：>1280px 时详情面板从 200 → 320px，可加更多元数据/预览

优点：单点 CSS 改动；大屏信息密度提升。缺点：中心网格可用列数略减

W3. 综合优化

详情

机制：W2 + 网格卡 110→140px + 详情面板元数据更丰富

优点：大屏体验最好。缺点：实现量稍大（多 token + DetailPanel 内容调整）
